Welcome to HiddenLayer's SDK documentation!
===================================

The HiddenLayer SDK for Python includes functionality to enhance the developer experience for securing AI and machine learniung models in Python.

It currently covers the following HiddenLayer services:

1. Model Scanner
2. MLDR


Installation
------------

Install the ``hiddenlayer-python`` package with `pip
<https://pypi.org/project/hiddenlayer-sdk>`_:

.. code-block:: console

    $ python -m pip install hiddenlayer-sdk

Check out the :doc:`getting-started.md` section for further information, including example usage.
.. note::

   This project is under active development.
